CITIZEN'S CHARTER

// SERVICE COMMITMENT

The Advanced Science and Technology Institute (ASTI) is committed to providing efficient, effective, and responsive services to all stakeholders. Our Citizen's Charter outlines the services we provide, the requirements needed, and the timeframe for each service delivery.

// OUR SERVICES

  • Research Grant Application

    Application for research grants and funding assistance

    5 working days Research Proposal, Budget Plan

  • COARE Computing Access

    Request for high-performance computing resources

    3 working days Project Description, User Requirements

  • PEDRO Data Access

    Request for satellite data and imagery

    7 working days Data Request Form, Project Description

  • DNS Registration (.edu.ph / .gov.ph)

    Domain name registration for educational and government institutions

    2 working days Organization Documents, Domain Request Letter
